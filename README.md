# 🚀 Piscine Reloaded

## 📝 Project Overview
A comprehensive collection of C programming exercises designed to reinforce core programming skills.

## 🔧 General Guidelines
- Compile with: `cc -Wall -Wextra -Werror`
- Follow Norm standards
- No forbidden functions
- Sequential exercise completion required

## 📚 Exercise Categories

### 🐚 Shell Scripting
- **ex00**: File/Directory Structure Management
- **ex02**: Advanced File Cleanup
- **ex03**: Shell Script File Discovery
- **ex04**: Network Interface Discovery

### 🧮 Basic Functions
- **ex06-08**: Alphabet & Number Printing
- **ex09-11**: Pointer Manipulation
- **ex12-14**: Mathematical Computations
  - Factorial (Iterative & Recursive)
  - Square Root Calculation

### 🔤 String Manipulation
- **ex15-17**: String Handling
  - String Output
  - Length Calculation
  - String Comparison

### 🚦 Program Control
- **ex18-19**: Argument Processing
  - Parameter Display
  - Argument Sorting

### 🧬 Advanced Techniques
- **ex20-21**: Dynamic Memory & Ranges
- **ex25-26**: Higher-Order Functions
  - Array Iteration
  - Conditional Counting

### 📦 Build & Configuration
- **ex22-24**: Macro & Makefile Development
  - Absolute Value Macro
  - Point Structure
  - Compilation Management
